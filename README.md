# COVID-Bot
QQ频道机器人-疫情助手

官方频道：[小念同学开发社区](https://qun.qq.com/qqweb/qunpro/share?_wv=3&_wwv=128&appChannel=share&inviteCode=1MVLD4&appChannel=share&businessType=9&from=246610&biz=ka
)

## 使用方法

```
/疫情 城市
    查询指定城市当天疫情数据
    示例： /疫情 深圳
/风险地区 城市
    查询国内风险地区
    示例： /风险地区 深圳
```
